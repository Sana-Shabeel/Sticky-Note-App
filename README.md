# Sticky Notes

A sticky note application built with react.js and css

## Description

A note taking application similar to sticky notes in design.

## Features
- Search for a note
- Delete a note
- Stores note in local storage
- Record a note using speech recognition API
- Edit A note
- Fully Responsive

## Built with

- React.js
- Css Grid
